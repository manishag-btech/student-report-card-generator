# Student Report Card Generator

A Python-based application that generates student report cards by calculating total marks, average, grades, and overall performance.

## Features
- Calculates total marks
- Calculates average and percentage
- Assigns grades
- Displays pass/fail status
- Identifies highest and lowest marks
- Shows best subject
- Provides performance analysis

## Technologies Used
- Python

## How to Run
1. Download the project files
2. Open the project in VS Code
3. Run student_report_card_generator.py
